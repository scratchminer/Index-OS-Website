FAQs

The perfect spot if you have some frequently-asked questions that need frequent answering.

What exactly is this?

Put simply: Index OS is a custom launcher for the Playdate game console! By installing it, you gain some nice functionality that you can't find in the stock Launcher, such as proper game sorting and folder management.

What's a Playdate?

If you don't know what a Playdate is, I'm not sure how you got to this page...but whatevz. Playdate is a video game console, made by Panic! You can find out more about it on Playdate's official website.

What features does Index have over the stock Launcher?

Lots of things! There's a whole list of 'em on the front page.

Why did y'all make this?

It initially started as just a want for some additional info on the Playdate's Launcher - things like the current time/battery and detailed game info - as well as easier folder-hopping. So, initially driven by that need for arguably useful info, we eventually wound up with Index!

Who made this?

All the code stuff was handled by scratchminer, and all the art stuff was handled by rae. You can find a full list of credits down at the bottom of this page!
