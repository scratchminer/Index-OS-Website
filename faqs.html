FAQs

The perfect spot if you have some frequently-asked questions that need frequent answering.

What exactly is this?

Index OS is a custom launcher for Playdate! Installing it nets you extra functionality that you can't currently get out of the stock game Launcher, such as a list view, in-depth game organization, and more.

What features does Index have over the stock Launcher?

Lots of things! There's a whole list of 'em on the front page.

Can I run this on actual hardware?

For sure! Just download the Installer file, and follow the setup process (carefully!!) to get up and running on Index OS.

Will this void my warranty?

On real hardware, yes. Per Playdate's Warranty page, they don't seem too fond of hardware that "has been intentionally modified using code (of any type, including malicious or otherwise)". Install at your own risk!!

Why did y'all make this?

It initially started as just a want for some additional info on the Playdate's Launcher - things like the current time/battery and detailed game info - as well as easier folder-hopping. So, initially driven by that need for arguably useful info, we eventually wound up with Index!

Who made this?

All the code stuff was handled by scratchminer, and all the art stuff was handled by rae. You can find a full list of credits down at the bottom of this page!

Okay, I downloaded it. How do I set this thing up?

Check out the Setup page for info on how to get everything started! Make sure to read everything carefully, and follow the correct steps for the platform you're using.

How do I switch back to the stock Launcher?

If you ever change your mind on Index, switching back to the stock Launcher is easy! Refer to the Setup page for steps on how to switch your default Launcher back to stock. Just make sure to follow the correct steps for the platform you're using.
