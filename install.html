Installation

The full guide on how to get Index OS up and running on your Playdate! It's as easy as 1, 2, 3.

On the Simulator

Here's a list of things you'll need before we begin:
• The Playdate Simulator (available at play.date/dev!)
• The Index OS .pdx file (Get it if you haven't already!)

Dropping it in

With the Index OS file in hand, navigate to path/to/PlaydateSDK/Disk/System. There, you'll find the .pdx for the stock Launcher - we strongly recommend keeping that file around in case you ever wanna switch back! Just rename it something else.

Next, drag the Index OS.pdx file into that folder, and rename it to 'Launcher.pdx'. Reset your sim, and you should be up and running on Index!

I wanna switch back!

You can switch your default back to stock at any time simply by coming back to this folder, and renaming the .pdx files. Just make sure the one you wanna use is named 'Launcher.pdx'!

On Real Hardware

  Wait a sec!
Installing Index on real hardware requires access to the Playdate's System partition, which is a very involved process that we won't teach how to do here.
There are a lot of risks that come alongside installing on real hardware, such as loss of warranty and potential for messing a lot of stuff up, like, really bad.
We're not responsible for any damage or loss that may come as a result of installing Index on hardware - so do this at your own risk!!

Here's a list of things you'll need before we begin:
• A computer! Any's fine, as long as you can use it to access the Playdate's disk.
• An unlocked Playdate console (that is, one that lets you access the System partition.)
• The Index OS .pdx file (Get it if you haven't already!)

Dropping it in

First, boot your console into System disk mode. If your Playdate's unlocked, you should know how to do this. Next, head into the System folder, located on the root of BERNARD. There, you'll find the .pdx for the stock Launcher - we strongly recommend keeping that file around in case you ever wanna switch back! Just rename it something else.

Next, drag the Index OS.pdx file into that folder, and rename it to 'Launcher.pdx'. Now eject your console, restart it if necessary, and you should be up and running on Index!

I wanna switch back!

You can switch your default back to stock at any time simply by coming back to the System folder, and renaming the .pdx files. Just make sure the one you wanna use is named 'Launcher.pdx'!
